# egress-check

检测各类网站的出口 ISP / ASN 信息，支持彩色终端可视化输出。

## 一键使用

```bash
curl -fsSL https://raw.githubusercontent.com/d93389894-cell/egress-check/main/egress-check.sh | bash
```

## 检测分类

- 🤖 Search & AI
- 💬 Social Media & Messaging  
- ▶ Streaming
- ₿ Crypto Exchanges
- 🛒 E-Commerce & Shopping
- 🌐 General
- 🔍 IP Test Sites

## 依赖

自动安装：`curl` `mtr` `bc`
